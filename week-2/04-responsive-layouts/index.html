Responsive Layout Techniques

Example 1: Fixed Layout

Fixed layouts use absolute units (pixels) and don't adapt to screen size.

Fixed Header (960px)
Fixed Sidebar (200px)

Fixed Main Content (760px)

This layout has a fixed width of 960px. It won't resize with the browser window.

Pros: Precise control over layout

Cons: May not fit all screen sizes, horizontal scrolling on small screens

Fixed Footer (960px)

Example 2: Fluid Layout

Fluid layouts use relative units (percentages) and adapt to screen size.

Fluid Header (100%)
Fluid Sidebar (25%)

Fluid Main Content (75%)

This layout uses percentages and adapts to the browser window size.

Pros: Adapts to different screen sizes

Cons: Less precise control, may become too wide or narrow

Fluid Footer (100%)

Example 7: Container with Max-Width

Combines fluid and fixed approaches for optimal readability.

Constrained Container

This container has:

  • width: 90% (fluid on small screens)
  • max-width: 1200px (prevents becoming too wide)
  • margin: 0 auto (centers the container)

This is a common pattern for modern responsive websites.

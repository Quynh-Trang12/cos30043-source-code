Example 3: box-sizing Property

The box-sizing property affects how total width/height is calculated.

content-box (default)
Width = content only
Total width = 200px + 20px padding + 5px border = 250px
border-box
Width = content + padding + border
Total width = 200px (includes everything)

Example 4: Margin Collapse
